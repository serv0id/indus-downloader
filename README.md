<p align="center">
  An APK downloader for the Indus App Store which replaced GetApps on Xiaomi phones.
</p>

## Installation

- `git clone https://github.com/serv0id/indus-downloader`
- Create a virtual environment
- `pip install -r requirements.txt`

## Usage

### Register a device
 - `python register.py`

### Search function
- `python main.py search [app name]`

### Download function
- `python main.py download [package name]`
